## Intended Use

一次性手术衣用于手术环境中为医护人员提供屏障保护，降低血液、体液和潜在污染物接触风险。

## Key Design Inputs

- 满足目标市场对手术衣分类和屏障性能要求
- 关键区域具备规定液体屏障能力
- 非关键区域兼顾透气性、舒适性和强度
- 袖缝、腰带、领口、袖口结构满足使用强度
- 包装和灭菌方式满足无菌供应要求
- 标签明确规格、灭菌状态、批号和有效期

## Candidate Verification Matrix

| Area | Candidate Test |
| --- | --- |
| Barrier | Hydrostatic pressure, impact penetration, synthetic blood penetration if applicable |
| Mechanical | Tensile, burst, tear, seam strength |
| Cleanliness | Particle, linting, bioburden where applicable |
| Sterility | Sterilization validation, routine monitoring |
| Packaging | Seal strength, integrity, transit simulation, aging |
| Usability | Donning/doffing, size coverage, sleeve and cuff fit |

## Initial Risks

- 液体穿透
- 接缝或袖口失效
- 穿脱过程污染
- 掉絮污染手术区域
- 包装破损导致无菌失效
- 规格或性能等级误用
