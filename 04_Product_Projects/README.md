# Product Projects

每个产品项目应建立独立文件夹，用于保存该产品的设计历史文件和关键证据链接。

建议命名方式：

```text
Product_Family_or_Model/
  README.md
  design-plan.md
  product-requirements.md
  risk-analysis.md
  standards-map.md
  verification-validation-plan.md
  verification-validation-summary.md
  design-reviews/
  test-reports/
  supplier-documents/
  packaging-sterilization/
  change-records/
```

项目文件夹不一定保存所有原始报告，但应能追溯到受控记录位置。
